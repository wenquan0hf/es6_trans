# 深度解析 ECMAScript 6 

ECMAScript 6 简称 “ES6”，ES6 包含了很多新的语言功能，且这些语言功能使得 JavaScript 更加强大更富有表现力。下面我们将会一步步介绍 ES6 新增的那些功能。

## 适用人群

本书为中级难度，适合已有一定 JavaScript 语言基础的读者，用来了解这门语言的最新发展；也可当作参考手册，查寻新增的语法点。

## 学习前提

学习本书前，你需要了解 JavaScript 这门语言。
